
# 🏐 Loja de Vôlei em Linguagem C

Projeto desenvolvido para a disciplina **Algoritmos e Técnicas de Programação I**, durante o 1º período do curso de Sistemas de Informação. A proposta da atividade consistia na criação de um software comercial utilizando os conceitos aprendidos em sala de aula e a linguagem C.

Por ser apaixonado por vôlei, decidi desenvolver uma loja virtual temática voltada para a comercialização de produtos relacionados ao esporte. O sistema simula operações básicas de uma loja, como consulta de produtos, controle de estoque, registro de compras e emissão de relatórios, permitindo aplicar na prática os conhecimentos adquiridos ao longo da disciplina.

## 🛠️ Conceitos Utilizados

Durante o desenvolvimento foram aplicados diversos conteúdos fundamentais da programação:

* Função `system()`
* Estruturas condicionais (`if`, `else` e `switch case`)
* Estrutura de repetição (`do while`)
* Matrizes multidimensionais
* Manipulação da Tabela ASCII
* Função `getchar()`

## 🎯 Objetivo

Aplicar conceitos de lógica de programação e desenvolvimento de software na construção de uma solução funcional, simulando um ambiente comercial de forma prática e contextualizada.

## 📈 Aprendizados

O projeto contribuiu para o fortalecimento de habilidades em lógica de programação, estruturação de algoritmos, organização de código e resolução de problemas computacionais.

## 🔮 Próximas Implementações

Embora o projeto já seja funcional, ele ainda está em fase inicial e servirá como base para futuras melhorias. Entre as possíveis evoluções estão a refatoração do código, implementação de novas funcionalidades, melhoria da experiência do usuário e utilização de estruturas mais avançadas à medida que novos conhecimentos forem adquiridos durante a graduação.
